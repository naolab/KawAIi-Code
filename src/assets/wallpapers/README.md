# 壁紙フォルダ

このフォルダにはアプリの壁紙ファイルが保存されます。

## フォルダ構成

- `default/` - デフォルト壁紙（背景1.png）
- `user/` - ユーザーがアップロードした壁紙
- このファイル - 使用方法説明

## 対応ファイル形式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- WebP (.webp)

## 推奨設定

- 解像度: 1920x1080以上
- ファイルサイズ: 5MB以下

## 使用方法

1. 設定画面から壁紙をアップロード
2. 一覧から壁紙を選択
3. 背景に自動適用されます
