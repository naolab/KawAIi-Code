使い方ガイド

基本操作

▶ 開始ボタン: AIアシスタント（Claude Code / Gemini Code Assist）を起動してチャットを開始します。
⏹ 停止ボタン: 起動中のAIアシスタントを停止します。
⚙ 設定ボタン（歯車）: 音声読み上げ、壁紙、作業ディレクトリ、キャラクター変更、プロジェクト固有設定などを変更できます。
? ヘルプボタン: この使い方ガイドを表示します。

AIアシスタント機能

💬 対話: 左側のターミナルに直接入力してAIアシスタントと会話します。
🔄 リアルタイム応答: AIアシスタントとのライブ通信で、スムーズな対話が可能です。
📝 動的プロンプト管理: AI起動時にキャラクター設定やプロジェクト固有の指示を.mdファイルとして自動生成・更新します。

キャラクター機能

🎭 3D VRMビューワー: 右側に3Dキャラクターが表示され、会話に応じてアニメーションします。
🗣️ 音声合成: AivisSpeech連携により、AIの応答が音声で読み上げられます。
👄 LipSync: 音声合成時にキャラクターの口が同期して動きます。
🚶 アイドルアニメーション: 会話がない間もキャラクターが自然な動きをします。

カスタマイズ

🖼️ カスタム壁紙: 設定から背景壁紙をアップロード・変更できます。時間帯別デフォルト壁紙も利用可能です。
📁 作業ディレクトリ: AIアシスタントが動作するフォルダを設定できます。

ヒント

⚡ 初回起動: 初めて使用する場合は、まず開始ボタンを押してAIアシスタントを起動してください。
🔌 音声エンジン接続: 音声エンジンに接続できない場合は、設定の『再接続』ボタンをお試しください。
